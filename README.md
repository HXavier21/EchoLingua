## 项目目标
1. 支持文本、语音、图像输入
2. 支持文本、语音结果输出
3. 翻译历史
4. 翻译设置
5. 用于测试的后端
6. 后端服务（翻译、语音合成、语音识别、OCR）交互
7. 预留离线模型使用接口
